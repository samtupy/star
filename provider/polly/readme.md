# polly

This provider wraps Amazon Polly cloud tts.

To use, you'll need an AWS account and a configured AWS cli, as well as pip install boto3. Learn more at https://aws.amazon.com/polly/

You'll need to run polly.py --configure to set your Amazon API key.

Rate is a multiple from 0.25 to 4, pitch is also a multiple with 1.0 being the default.
